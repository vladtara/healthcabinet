# Presentation Assets

This folder contains the current HealthCabinet pitch presentation materials.

## Files

- [`healthcabinet-pitch-deck.pptx`](healthcabinet-pitch-deck.pptx)  
  Native presentation file for PowerPoint, Keynote import, and standard slide-sharing workflows.

- [`healthcabinet-pitch-deck.html`](healthcabinet-pitch-deck.html)  
  Standalone designed deck for browser presentation, screen sharing, or PDF export through the browser print dialog.

- [`healthcabinet-pitch-deck.md`](healthcabinet-pitch-deck.md)  
  Slide-by-slide outline with speaker notes, positioning language, and the intended story arc.

- [`generate_pitch_pptx.py`](generate_pitch_pptx.py)  
  Local generator for rebuilding the `.pptx` deck from the current presentation content and design language.

## Recommended Use

- Use the HTML deck when you need the visual presentation.
- Use the PPTX deck when you need to present in PowerPoint-compatible tools.
- Use the Markdown deck when you are tailoring the story for a specific audience, editing headlines, or preparing speaker notes.
- Replace the mock dashboard/table visuals with real product screenshots as the UI stabilizes.

## Narrative Frame

The current deck is optimized for:

- investor-style pitch conversations
- partner demos
- design or product reviews where the audience needs the product story quickly

Core positioning:

> HealthCabinet turns fragmented lab documents into a longitudinal health intelligence product people can understand and trust.
